Ubuntu 桌面 - 应用软件

Dec 22, 2021 家猫

Ubuntu 桌面系统有很多实用的应用软件，通常是和 Windows 系统下的应用软件是对标的；Ubuntu 软件仓库提供的程序的最大特点，也可以说是优点，就是绝大多数是开源的。

Atom 编辑器

Atom 是 Github 专门为程序员推出的一个跨平台文本编辑器，丰富的插件几乎能够满足所有 web 开发需求。

从官网 安装文件列表 或者 华为云镜像网站 下载安装文件atom-amd64.deb到本地，并点击安装。

KeePassXC 密码管理器

KeePassXC 是著名的密码管理器 KeePass 的一个分支，是一个操作简单，并且自带中文界面的免费密码管理器。

1sudo add-apt-repository ppa:phoerious/keepassxc </br>
2sudo apt update </br>
3sudo apt install keepassxc </br>

Ubuntu Cleaner

Ubuntu Cleaner 是一个系统管理工具，有着易于使用的简单用户界面，用来移除不再使用的包、不需要的应用和清理浏览器缓存。

1sudo add-apt-repository ppa:gerardpuig/ppa </br>
2sudo apt update </br>
3sudo apt install ubuntu-cleaner </br>

Uget 下载管理器

Uget 是一款优秀的下载管理工具，可以提高下载速度。

安装 uget 和 aria2

1sudo add-apt-repository ppa:plushuang-tw/uget-stable </br>
2sudo add-apt-repository ppa:t-tujikawa/ppa </br>
3sudo apt-get update </br>
4sudo apt-get install uget aria2 </br>

配置 Uget：打开安装好的 Uget，在菜单上依次选择 Edit > Settings > Plug-in > Plug-in matching order > aria2，将 aria2 配置为优先。

安装 uget-integrator

1sudo add-apt-repository ppa:uget-team/ppa </br>
2sudo apt update </br>
3sudo apt install uget-integrator </br>

加载插件

以扩展形式加载到 Firefox 浏览器

  • 打开 Firefox 浏览器，找到并点击右上角 Menu，选择 Add-ons；
  • 在 Add-ons 的界面的 Find more add-ons 中搜索 uget；
  • 在结果列表中选择 uGet Integration，点击 Add to Firefox 按钮即可。
